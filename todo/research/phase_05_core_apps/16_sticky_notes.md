# 77 — Sticky Notes

## Overview
Desktop sticky notes — quick text notes that float on the desktop. Like Windows Sticky Notes.

## Features
- Click "+" to create new note
- Resizable colored rectangles with text
- Auto-save to Codex
- Color options: yellow, pink, blue, green, purple
- Notes persist across reboots

## Data: `User\{name}\StickyNotes\{id}\Text`, `...\Color`, `...\X`, `...\Y`, `...\W`, `...\H`
