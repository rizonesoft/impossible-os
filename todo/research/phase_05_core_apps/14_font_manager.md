# 69 — Font Manager

## Overview
App to view installed fonts, preview text, install/remove fonts. Fonts stored in `C:\Impossible\Fonts\`.

## Features
- List all installed `.ttf`/`.otf` files
- Preview each font at different sizes: "The quick brown fox..."
- Install new fonts (copy `.ttf` to fonts directory + register in Codex)
- Remove fonts
- Set default system/monospace font
